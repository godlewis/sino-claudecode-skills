# Hello World Skill

这是一个简单的示例技能包，用于演示如何创建Claude Code技能。

## 功能

当用户说"hello"或"你好"时，以友好的方式回应。

## 使用示例

用户输入:
```
hello
```

Claude回应:
```
你好！我是Hello World技能。很高兴为您服务！
```

## 技能结构

这是一个最小化的技能包示例，包含：

- **SKILL.md**: 技能定义文件（必需）
- **README.md**: 使用说明
- **CHANGELOG.md**: 更新日志

## 创建自己的技能

1. 复制此技能目录作为模板
2. 修改SKILL.md中的name和description
3. 添加技能的具体逻辑和说明
4. 测试技能是否正常工作

## 最佳实践

- 保持description清晰简洁，说明技能的使用场景
- 使用markdown格式编写文档
- 提供具体的使用示例
- 保持技能简单、专注
